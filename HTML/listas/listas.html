Tipos de listas HTML

Listas ordenadas:

As listas ordenadas elas usam números para identificar os itens desta lista.

  1. 1º item da lista
  2. 2º item da lista
  3. 3º item da lista
  4. 4º item da lista

Listas não ordenadas:

As listas não ordenadas usam marcadores para identificar os itens

  • Abacaxi
  • Maça
  • Pessego

Listas de definição:

HTML
Linguagem de marcação de Hipertexto usada para criar paginas na web.

CSS
Folhas de estilo em cascata, usadas para estilizar a aparência de elementos HTML.

JavaScript
Linguagem de programação utilizada para tornar as páginas web interativas.
